## 本集主題（含 emoji 意象解讀）

**🐸 青蛙意象**：青蛙在水中靜靜蟄伏、伺機跳躍，呼應本集「市場在壓力位蓄勢、等待新族群接棒」的主旋律。CPU 題材、被動元件正如青蛙起跳前的積蓄，資金悄悄轉移但市場尚未全面定價。

本集核心軸線：**AI 算力缺口從 GPU 蔓延到 CPU，被動元件缺貨潮再起，選擇比努力更重要。**

## 一、重點摘要

### 1. 市場定位與族群輪動

- 台股拉至歷史新高附近；美股雖站回季線，但上方壓力仍需消化
- 中東事件影響已鈍化，邊際效應下降，市場選擇無視
- Q1 問題不在缺乏強部位，而在同時持有**表現落後的晶片、巨頭與軟體股**，導致淨值被拖累
- Q2 資金重新調配：從光通訊單線轉向 **CPU、Marvell、高速傳輸、被動元件**多族群聯動
- 謝孟恭觀點：「**方向選對，比單純努力操作更重要**」——選錯標的，再怎麼用功也難逆轉

### 2. CPU 成為 AI 算力新主戰場

- CPU 已從「題材討論」轉為「真金白銀驗證」，部位上漲速度加快
- Agentic AI 時代：每 1 GW 代理式 AI 算力需要傳統 AI 訓練叢集 **4 倍以上** 的 CPU 核心
- CPU 角色不只配角，更像「**教練**」——負責調度、協調、分配任務給 GPU
- Amazon CEO Andy Jassy：「Agentic AI 正在成為幾乎和 GPU 一樣重要的 CPU 故事」
- **AWS Graviton 關鍵訊號**：兩家大型 AWS 客戶要求「包下 2026 年全部 Graviton 算力」，AWS 拒絕，因自身都不夠用 → 需求強度遠超市場預期
- Meta 簽下數十億美元、多年期的 AWS Graviton5 合約，專門用於代理式 AI 推理工作負載
- **Nvidia 投資 SiFive（RISC-V 架構）**：市場重新關注特化 CPU 潛力，視線從「純 GPU」擴展至 RISC-V、新 CPU 架構

**台股相關標的**：晶心科（RISC-V IP 核）、聯發科、創意、世芯-KY
**美股相關標的**：Nvidia、Amazon、Qualcomm、Synopsys、Cadence、Marvell、Broadcom

### 3. AI 模型算力不足的真相：Anthropic Mythos

- Anthropic 發布 **Mythos Preview**（2026/4/7），以「太強、太危險」為由限制開放
- 謝孟恭解讀：**真實原因是算力根本撐不住全面開放**——一旦釋出，用量爆增、整體體驗崩潰
- 白宮反對擴大 Mythos 授權，CNBC 指出「Anthropic 根本沒有足夠算力擴大部署」
- **「AI 變笨」現象**：非模型退步，而是美東白天高峰上線人數太多、算力不足所致
- **「末日行銷」本質**：廠商用「模型太強危險」敘事掩蓋供給不足，副作用是引發「泛軟體股被取代」恐慌
- Cloudflare、Palantir、CrowdStrike 等原本有機會重爬的軟體股再被情緒打壓

### 4. 軟體股暫非主戰場

- 軟體股基本面並未否定，但資金效率更高的是**硬體、算力、傳輸、CPU**
- 現階段多族群聯動（CPU＋光通＋高速傳輸＋先進封裝）比 Q1 單靠光通撐場更健康
- 謝孟恭：「**軟體股不是不能看，而是現在資金不在那裡**」

### 5. 被動元件缺貨潮再起（類 2017 結構）

- 謝孟恭 2-3 月前已布局，當時太早認輸；現重新進場，產業訊號比前期清楚
- **缺貨品項與嚴重程度**：
  - 鉭電容（T-Cap）：最嚴重，寡占供給 + AI 伺服器需求爆量；國巨已於 2 月啟動漲價
  - 鋁固態電容（SPCAP）：AI 主機板、高速運算需求拉動
  - MLCC：高階品項接近警戒線，交期已拉長至 22-24 週；村田漲價 15-20%
- 單台 AI 伺服器被動元件需求較傳統伺服器多 **50%-500%**
- **結構類比 2017 年**：當年手機＋電動車拉動；現在換 AI 伺服器當主角
- **催化劑**：**國巨法說會（2026/4/15）** 為本週最重要觀察點——預期 lead time 上揚、展望偏正面
- 事後驗證：國巨法說會釋出正面指引，毛利率目標挑戰 38.6%，Q2 樂觀看待

**台股相關標的**：國巨（2327）、華新科（2492）、信昌電（6173）、立隆電（2472）、台慶科（2496）、華泰（2329）

### 6. 光通訊擴產「借廠房、借產線」新模式

- 光通訊需求太強，自有廠房擴產速度跟不上（環評、土地、建廠耗時）
- 新模式：**美系大廠自備設備，進駐台灣閒置 LED 廠 / 網通廠 / SMT 廠產線代工**
  - 不是純代工下單，而是「設備＋訂單一起導入」
- 台股已有約 **4 家公司**進入此模式（未點名），代表現象已擴散而非個案
- SMT 打件毛利偏低（約 10% 以下），但對原本稼動率不足的廠商，大單也能改善獲利
- 謝孟恭提醒：「**這題材偏深水區，不能太早亂買**」——先撈名單、觀察股價轉強再決策
- 風險：客戶今日導入設備，日後隨時可抽走
- 偵測方向：SMT 產線稼動率偏低的閒置廠商（譁裕、弘凱、艾笛森、台表科、聯嘉等方向）

## 四、投資心法與金句

1. **「方向選對，比單純努力操作更重要。」**
   選錯族群，再精準的買賣點也難逃落後；Q1 的教訓是「抱光通訊卻同時壓著爛股」。

2. **「CPU 像教練，不只是配角——負責調度、協調、分配資源。」**
   GPU 是球員，CPU 才是讓球員發揮的體制核心。

3. **「已有客戶想直接全包 Graviton 算力，但 AWS 自己都不夠用。」**
   CPU 需求強度遠超市場原本預期，瓶頸已從 GPU 擴散。

4. **「AI 變笨，往往是高峰時段被擠爆，不是模型退步。」**
   性能下滑≠技術倒退；是供給跟不上需求的現象。

5. **「軟體股不是不能看，而是現在資金不在那裡。」**
   基本面未崩壞，但資金輪動暫時離場，等敘事重構。

6. **「結構有點像 2017 年，只是主角換成 AI 伺服器。」**
   被動元件缺貨的驅動力換了，邏輯相同：高端應用抽乾產能，其他品項出現缺口。

7. **「這題材偏深水區，不能太早亂買。」**
   光通訊借廠模式：概念成立，但名單難判斷，需等催化訊號再進場。

8. **「股價創歷史新高，反而是買進介入的時機。」**（Threads 筆記）
   挑戰「逢高拋售」直覺；歷史新高背後若有強基本面故事，往往才是起點。
