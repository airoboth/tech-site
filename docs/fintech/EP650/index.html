股癌 Gooaye EP650 | 🦄

播出日期：2026 年 4 月 4 日（星期六）｜時長：50 分鐘｜主持人：謝孟恭
平台：Apple Podcasts、Spotify、SoundOn、YouTube
開場金句：「見面第一句話就是 OGC（Optics, GPU, CPU）」

本集主題（含 emoji 意象解讀）​

🦄 獨角獸（Unicorn），在科技與創投界象徵「估值超過 10 億美元的私人新創企業」，也代表稀有與顛覆。本集以獨角獸為題，呼應 Agentic AI 時代的「CPU 地位逆轉」——這個在 GPU 年代被邊緣化的元件，在 Agentic AI 工作流中卻意外成為最大效能瓶頸，一夕之間從配角躍升為「獨角獸級」投資標的。

開場三字咒語 OGC（Optics、GPU、CPU） 奠定本集三大技術軸線，象徵光學、運算架構與處理器三條投資主線的交匯。

一、重點摘要​

1. 個人分享：瘦瘦針（GLP-1 針劑）使用心得​

  • 謝孟恭分享第二針後身體反應，食慾下降約 三成（第一針僅 10–15%）
  • 體重計追蹤：掉落的主要是肌肉量，脂肪維持良好
  • 強調持續運動的重要性，以防止肌肉流失
  • 醫師告知可能出現性慾下降副作用
  • 搭配 OMRON 八電極體脂計追蹤身體組成變化

2. 地緣政治：荷莫茲海峽美伊衝突​

  • 美國在荷莫茲海峽遭遇空戰，損失 F-15 戰機；一名飛行員已被救回，武器官（WSO）仍在搜索中
  • 五角大廈「披薩指數（Pizza Tracker）」：觀察國防部周邊披薩外送訂單量異常飆升，作為美軍緊急行動的另類指標
  • 謝孟恭觀點：「短期承受油價上漲，換取未來不確定性降低，是划算的交易」
  • 核心投資心法：「猜對事件不是重點，事發生之後怎麼應對，才是重點」

3. CPU 族群：Agentic AI 引爆 CPU 價值重估（本集核心）​

背景轉變​

AI 從對話式（ChatBot）進入 Agentic AI 時代——AI 代理可自主規劃子任務、查詢資料、呼叫 API、執行程式碼，形成反覆迭代的閉環工作流。

CPU 瓶頸論（學術研究支撐）​

維度傳統 AIAgentic AI
CPU 佔總延遲~10%（Cornell 大學）50–90%（Georgia Tech 論文）
CPU:GPU 部署比1:4 ~ 1:8趨向 1:1 ~ 1:2
每 GW CPU 核心需求3,000 萬（ARM 估算）1.2 億（4 倍，ARM 估算）
CPU 能耗佔比較低動態能耗達 44%（大批次）

各公司動態與謝孟恭觀點​

AMD

  • Lisa Su 指出 agent 工作流仍依賴傳統 CPU 任務
  • Q4 2025 資料中心營收 $53.8 億，年增 34%
  • 2026 年 4 月股價因 Agentic AI 題材單日上漲 14%，D.A. Davidson 給予 Buy 評級，目標價 $375

Intel

  • CFO 坦言「misjudge the pace of demand」
  • 伺服器產能「absolutely constrained（完全受限）」
  • Q1 2026 DCAI 營收 $51 億，年增 22%
  • CEO Lip-Bu Tan：AI 正從「基礎模型 → 推論 → Agentic」演進

ARM Holdings

  • 伺服器市佔從 2018 年 <5% → 2025 年 20–25%
  • AWS Graviton5（192 核心）、Google Axion、Azure Cobalt 200 全面採用 ARM 架構
  • 2026 年 YTD 漲幅高達 +87.18%，ARM 目標搶下資料中心 CPU 市場 50% 份額

NVIDIA

  • Jensen Huang 提及未來更多獨立 CPU 部署
  • GB200 搭載 ARM-based Grace CPU 核心（ARM 生態延伸）

嘉澤（3533）

  • 台灣 CPU 插槽（Socket）與高速連接器龍頭
  • 受惠伺服器平台升級至 6096 Pin、高速傳輸規格提升
  • AI server + 液冷快速接頭貢獻下半年業績

操作策略​

  • 優先選 AMD 與 Intel（「純度高」的 CPU 純股）
  • 台股優先關注嘉澤（3533）等 CPU socket 與高速連接器概念
  • 採趨勢動能策略：越漲越買；跌破支撐則停止加碼，而非估值導向

4. 鋁價上漲與被動元件遷徙​

  • 中東兩座鋁廠遭轟炸，影響全球鋁供應約 15%
  • LME 鋁錠庫存降至 397,100 噸，市場由供需偏緊轉為結構性短缺
  • 鋁價創歷史新高，連帶推升鋁電容（Aluminum Electrolytic Capacitor） 缺貨預期

替代方案分析：

元件說明代表廠商
MLCC（積層陶瓷電容）微小儲能裝置，穩定電路電流，取代鋁電、鉭電三星電機、村田、國巨
TLVR（穿頂式傳輸線電壓調節模組）極速電壓調節，因應算力功耗暴增電感廠商
  • 謝孟恭定性此主題為「時機財」與「事件財」：短期有缺貨漲價機會，但 MLCC 廠長期 EPS 本益比擴張空間有限
  • 結合 Vera Rubin（NVIDIA 下一代 GPU 平台）大規模建置需求，短期題材可期
  • 被動元件供應商 Q1 2026 整體漲價 10–15%，MLCC B/B Ratio 從 3 月 0.89 升至 4 月 0.92

5. 塑化股觀察​

  • 友人邏輯：美伊衝突封鎖伊朗石油 → 中國失去廉價原油來源 → 中國塑化業競爭力下降 → 台灣塑化股受惠
  • 謝孟恭分享此觀點供聽眾參考，未提供明確操作建議

6. 聽眾問答精選​

  • 做空 vs 做多：做多期望值理論無限；做空最多賺 100%。長期空頭若持續虧損，應順應市場轉向做多
  • 基本面研究：新手常在「很淺的地方」，需歷經多個景氣循環才能真正理解供需變化
  • 職涯困境：若工作能滿足當前財務需求，先忍耐、待機轉換，不必急於一時

四、投資心法與金句​

  1. 「猜對事件不是重點，事發生之後怎麼樣去做處理，那才是重點。」 — 強調反應能力優於預測能力

  2. 「基本面研究的本質是找預期差。」 — 市場已知的不值錢，關鍵在發現市場尚未定價的認知落差

  3. 「買進理由要清楚：價值、動能、還是題材？」 — 三種框架對應不同進出場邏輯，混淆則會兩頭落空

  4. 「避免成為死空頭：做多期望值無限，做空最多賺 100%。」 — 長期做空的數學期望值天生不對稱

  5. 「股票終究要回歸供需，最終還是要猜摩一下供需的變化。」 — 再複雜的主題，落地還是在供需缺口

  6. 「物質只是追求的一小部分；羨慕嫉妒破壞遊戲體驗；幸福在單純之物。」 — 投資心態：學會欣賞自己的進步，不與他人比較

  7. 「認知程度的差別，決定了你能看見的機會。」 — 研究深度決定投資勝率
