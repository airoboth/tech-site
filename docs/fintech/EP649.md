## 本集主題（含 emoji 意象解讀）

**💌 太…太刺激了…**

「💌」是一封情書信封，象徵 NVIDIA 以 20 億美金投資 Marvell 的「告白式結盟」——在 ASIC 戰場你爭我奪之際，黃仁勳主動遞出情書、四處插旗，拉攏盟友對抗博通。同時，台股讓多頭興奮，美股卻讓人捏冷汗；這種「又甜又驚」的矛盾感，正是本集「太刺激了」的雙重解讀。

本集三大核心：
1. **台美股分歧**：台股多頭強勢 vs. 美股死貓跳，操作策略大不同
2. **NVIDIA 結盟 Marvell**：20 億投資背後的 ASIC 戰場底層邏輯
3. **玻璃基板時程與光通訊佈局**：題材卡位期的操作心法

## 一、重點摘要

### 1. 閒聊：財富自由後的生活觀

- **瘦瘦針（減重針）心得**：食量明顯下滑，零食慾望幾乎消失，效果顯著
- **新運動：網球**：報名教練課，每堂費用 1,700 元，親身體驗「買健康」的價值
- **核心體悟**：
  > 「工作再怎麼做對生活影響不大，但健康沒顧好對生活影響才真的很大。」

達到財務自由後，主委認為應將資源優先投入健康與生活體驗，而非繼續衝刺工作。

### 2. 盤勢觀點：台股多頭 vs. 美股死貓跳

**台股**
- 回踩重要均線後強勢反彈，若後續持續創高，確立多頭回檔格局
- 融資雖下滑，但未出現全面恐慌殺盤，籌碼仍算乾淨

**美股**
- 主委視為「死貓跳」，已提前降低美股部位水位
- 兩大風險指標持續高掛：
  - **油價**：近月、遠月皆未回落，WTI 高位，歷史上油價衝過百美元對股市不利
  - **VIX**：仍在高位震盪，波動率尚未收斂，隨時可能造成破壞性修正

**操作結論**
- 全面轉向**純現股操作，零槓桿**
- 部位採「分層次調節」，保留現金作為戰略縱深
- 避免 All-in / All-out 的零和思維

### 3. NVIDIA 結盟 Marvell：ASIC 戰場的底層邏輯

**事件**：2026年3月31日，NVIDIA 宣布投資 Marvell **20 億美元**，雙方以 NVLink Fusion 平台深化合作。

**NVLink Fusion 是什麼？**
- 一種架構平台，讓異質晶片（客製 XPU、ASIC）能夠以高速無縫接軌 NVIDIA 生態系
- Marvell 提供客製 XPU 及相容的高速網路方案
- NVIDIA 提供：Vera CPU、ConnectX NIC、BlueField DPU、NVLink 互聯、Spectrum-X 交換器

**謝孟恭解讀：三層戰略意圖**

| 層次 | 說明 |
|------|------|
| 表面 | 「錢太多去買東西而已」——NVIDIA 現金充裕，戰略配置 |
| 中層 | 藉 Chiplet、Grace CPU、Switch、CPO 等「周邊設施」掌控 CSP 自研 ASIC 趨勢下的資料中心話語權 |
| 核心 | **真正對手是 Broadcom**——在 Google、AWS、Meta 客製晶片浪潮下，NVIDIA 搶在博通前拉攏 Marvell 等廠商，分食 ASIC 資料中心利潤 |

**對相關個股的影響**
- Marvell (MRVL)：受益最直接，但「宣示意味大於實質利多」，需觀察後續發酵時機
- Broadcom (AVGO)：感受最大壓力，市場份額受蠶食風險上升
- Astera Labs (ALAB)：與 Marvell 在伺服器架構中位置不同，可共存受益，無被取代危機
- Alchip、MediaTek（發哥）：在小型晶片商結盟陣營中仍有角色

### 4. 玻璃基板：2028 年前別急著數鈔票

**技術簡介**

| 術語 | 說明 |
|------|------|
| GCS（Glass Core Substrate） | 以玻璃取代傳統有機材料做為晶片封裝基板，散熱與精度更佳 |
| TGV（Through Glass Via，玻璃通孔） | 貫穿玻璃基板的垂直導電通道，是 GCS 的核心製程 |

**量產時程**
- 主委判斷：**GCS 與 TGV 真正放量要等到 2027 中〜2028 年**，2028 年後才有大量
- Intel 是最早推動玻璃基板的大廠，在台灣合作夥伴開發上走在前面
- 設備股：約半年後才可望看到明確營收貢獻

**操作策略類比**
- 類似 CPO（共同封裝光學）的前置行情：CPO 約 2026 年出量、2027 年大量，但相關股票在那之前兩三年就已經在動
- 目前市場資金是「**卡位性質**」的提前佈局，並非實質營收驅動
- 主委建議：操作上應視為投機題材，**嚴設停損停利**，不宜全倉押注

## 四、投資心法與金句

**部位控管**
> 「部位是可以分層次調節、具備戰略縱深的。」——避免 All-in 或 All-out 的零和思維

> 「採取分批加減碼方式，保留現金水位作為戰略縱深。」

**心態管理**
> 「每次的修正其實某個層面上都是讓你買進的機會，但怎麼控好自己的部位是一個很大的修煉。」

> 「最怕的就是心態卡住，做出不理性的判斷跟情緒化的操盤。」

**交易記錄**
> 「把你的交易給記錄下來，因為很多時候你的感覺會騙你。」

**市場容量觀**
> 「就算被吃掉兩根手指，整個拳頭還是可以變更大。」——市場不是零和遊戲

**健康哲學**
> 「工作再怎麼做對生活影響不大，但健康沒顧好對生活影響才真的很大。」

**槓桿原則**（本集重申）
- 全面轉向現股，不上任何槓桿，避免心理壓力與斷頭風險
- 「戰事擴大前市場仍有消化空間，但融資清潔度不足，反彈後仍需小心。」
